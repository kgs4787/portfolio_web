# 김강산 포트폴리오 웹사이트

개인 포트폴리오 웹사이트입니다. React, TypeScript, Tailwind CSS를 사용하여 개발되었습니다.

## 🚀 주요 기능

- 반응형 디자인
- 스크롤 기반 네비게이션
- 애니메이션 효과
- 접근성 최적화

## 🛠 기술 스택

- **프론트엔드**
  - React
  - TypeScript
  - Tailwind CSS
  - Framer Motion (애니메이션)
  - React Scroll (스크롤 네비게이션)

## 📁 프로젝트 구조

```
src/
├── components/
│   ├── atoms/      # 기본 컴포넌트
│   ├── molecules/  # 복합 컴포넌트
│   └── organisms/  # 페이지 섹션 컴포넌트
├── pages/          # 페이지 컴포넌트
└── styles/         # 전역 스타일
```

## 🚀 시작하기

### 필수 조건

- Node.js (v14 이상)
- npm 또는 yarn

### 설치

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 🔧 빌드

```bash
# 프로덕션 빌드
npm run build

# 빌드 결과물 실행
npm run start
```
